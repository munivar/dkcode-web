How to Enable Floating Search using Registry Editer (Immersive Search)

1. Search and Run Registry Editor.

2.Now Paste this Following Path to Registry Editor Address Bar...

Copy Path

3. Right click on "Search" folder and create a new DWORD 32-bit key Called "ImmersiveSearch".

4. Double click on "ImmersiveSearch" and set value data "1".

5. Now Expand Search Folder and select "Flighting" Folder.

6. Right click on "Flighting" folder and Create New Key Called "Override".

7. Now Right Click on Override Folder and Create New 32bit DWORD Key Called "ImmersiveSearchFull".

8. Double click on "ImmersivSearchFull" and Make sure that value data is "0".

9. Restart your Computer.

7. And It's Done.

--- --- --- --- --- --- --- ---

 

Back to Default

1. Follow 1 and 2 step. Delete "Override" folder.

2. Delete "ImmersiveSearch" key.

3. Restart your Computer.
4. And It's Done.

How to Bypass TPM 2.0 and Secure Boot cheaking when Installing Windows 11

After Launching Windows 11, Microsoft Set Some Compatibility in Windows 11. Some Old PC like My Computer not fill this Compatibility. So, What can we do? The Answer is Run Windwos 11 without cheaking TPM 2.0 and Secure Boot.

1. Download Windows 11 .iso file and Create a Bootable Pendrive Using Rufus. Rufus is helping to create a Bootable Pendrive. Cheak MBR and GPT Partition Style and Create Bootable Pendrive using Windows 11 .iso file.

2. Before Booting Pendrive in your System. Make sure to Backtup All your Important Data.

3. Boot Pendrive to Your PC. When boot screen Appear, Press (Shift+F10). It will open "Command Prompt".

4. Type "regedit" in CMD to Open Registry Editor.

5. Go to [ HKEY_LOCAL_MACHINE > SYSTEM > Setup ]

6. Now Right click on "Setup" folder and Create a new key called "LabConfig".

7. Right click on "LabConfig" and Create a New DWORD 32-bit key called "BypassTPMCheak".

8. Double click on "BypassTPMCheak" and set Value data "1" and click on "OK".

9. Now Right click on "LabConfig" and Create a New DWORD 32-bit key called "BypassSecureBootCheak".

10. Double click on "BypassSecureBootCheak" and set Value data "1" and click on "OK".

11. Close Registry Editor and Command Prompt.
12. And you are Good to go for Installation.

Back to Default

1. Search and Run Registry Editor in your windows 11 installed Computer.

2. Go to [ HKEY_LOCAL_MACHINE > SYSTEM > Setup ] and Delete "LabConfig" folder.

3. Restart your PC and It's Done.

How to Add Any Application in Windows Context Menu (Right Click Option) (Chrome Browser)

The Registry Key allows you to have a Custom Application on the Right-Click Context Menu. For instance, You can put up Calculator, Notepad, or WhatsApp on the Right-Click Context Menu. I like to have it on the Right-Click Menu for Convenience.

1. Search Registry Editor And Run as Administrator

2.Now Paste this Following Path to Registry Editor Address Bar...

Copy Path

3. Right Click on "Shell" folder and Create a New Key Called "Chrome".

4. Right Click on "Chrome" folder and Create a New Key Called "Command".

5. Now Search Chrome in Start Menu, Right Click on Chrome And Click to "Open File Location".

6. It will Redirect you to File Explorer. Now Right Click on "Chrome.exe" and Click on "Open File Location".

7. Now, In File Explorer, Go to Home Tab and Select "Copy Path".

8. Go Back to Registry Editor, On Right Side, Double Click on "Default". And Paste the Copied Path in "Value Data". And Click on "OK".

9. It's Done.

How to Add Chrome Incognito Tab in Windows Context Menu

I use it Slightly Different. I used the Chrome Incognito Tab too often for Research Purposes and I like to have it on the Right-Click Menu for Convenience.

1. Follow Above 1 and 2 Step.
2. Right Click on "Shell" folder and Create New Key Called "Chrome Incognito".
3. Now, Right Click on "Chrome Incognito" folder and Create New Key Called "Command".

4. Now Follow Step 5,6 and 7.
5. Right Side, Double Click on "Default" and Paste the Copied Path. And set value Data and Make sure to Write "--Incognito" after Path. Click on "OK"

6. It's Done.

How to Remove Any Application from Windows Context Menu

1. Follow 1 and 2 Step and Delete the "Chrome" Folder.

2. Confirm your Key Deletion and Click "Yes".

3. It's Done.

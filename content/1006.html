How to Add CopyTo-MoveTo option in Windows Context Menu (Right Click Option)

Some People like me is upset with Windows Copy-Paste Option. like open one windows and copy file and open paste location. it's time Consuming process, is't it? so, I will Add CopyTo-MoveTo Shortcut in my windows context menu.

1. Search and Run Registry Editor.

2.Now Paste this Following Path to Registry Editor Address Bar...

Copy Path

3. Right click on "ContextMenuHandlers" and Create a new key called "CopyTo".

4. On Right side Double click on "Default" and Set Value Data
"{C2FBB630-2971-11D1-A18C-00C04FD75D13}" and click on "OK".

5. Right click on "ContextMenuHandlers" and Create a new key called "MoveTo".

6. On Right side Double click on "Default" and Set Value Data
"{C2FBB631-2971-11D1-A18C-00C04FD75D13}" and click on "OK".

7. Restart your Computer.

Note: To Cheak it Works on Not, Please Right click on Any file/Folder.

8. It's Done.

Back to Default

1. Follow step 1 and 2.
2. Delete CopyTo and MoveTo folder.

2. Restart your Computer.

3. It's Done.

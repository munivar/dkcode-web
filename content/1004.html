How to Permanently Set Desktop Background in Windows

1. Search and Run the Registry Editor.

2.Now Paste this Following Path to Registry Editor Address Bar...

Copy Path

3. Right click on "Policies" folder and Create a new Key called "System".

4. Right click on "System" folder and Create new String Value called "Wallpaper".

5. Now Copy your Desktop Wallpaper Image Path. In my case my wallpaper is in Download Folder. so, i can copy my Image like this ...
(Select image and Under home tab click on "Copy Path".

6. Go back to Registry Editor. Double click on "Wallpaper" and Under the Value Data , Paste the Image Location.
Make sure to remove ("")Cortation Mark and Click on "OK".

7. Restart your Computer
8. And It's Done.

Back to Default

1. Follow 1 and 2 Step and Delete "System" folder.
